# WindowBlessing

Forget Curses! Try Blessings instead! WindowBlessing is an evented windowing framework for terminal apps.

## Installation

Add this line to your application's Gemfile:

    gem 'window_blessing'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install window_blessing
